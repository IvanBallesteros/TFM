## Repository Structure

### `Models/`
Stores all trained models and their different versions.

### `Model_Notebooks/`
Contains the development notebooks for:
- The **CNN-based Computer Vision model**
- The **Text Classifier using OCR**
- An **Exploratory Data Analysis (EDA)** notebook, including dataset insights and data splitting methodology  

These notebooks make use of shared utility functions located in the `Utils/` folder.

### `Utils/`
Includes helper functions and reusable components used across the notebooks.

### `Inference/`
Contains notebooks for:
- Running inference on each model independently  
- Implementing the **multimodal approach**, combining both models  
- Performing validation using the test set also in the Multimodal notebook
